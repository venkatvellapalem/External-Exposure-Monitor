Decoupled Attack Surface Architecture

Learn how EASM monitors threat exposure with zero target traffic overhead.

Core Principles & Design Pillars

01

Passive Reconnaissance

Queries Shodan InternetDB & Censys Platform v3 API host lookups silently without sending a single probe packet to target servers.

02

Surgical Active Verification

Performs targeted, non-intrusive 2.5-second Python TCP handshakes only on candidate exposed ports to eliminate false positives.

03

Sensitive File Leak Audit

Audits open web services (80/443) for exposed configuration files (/.env, /.git/HEAD) using host header routing.

04

Stateful Reconciliation

Tracks port state lifecycles in local baseline memory, automatically sending status="closed" alerts to Splunk when risks are fixed.
